# Earthquake EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on earthquake data to understand patterns related to tsunami and non-tsunami events.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning
- Histogram and box plot analysis
- Scatter plot for latitude and longitude
- Heatmap for correlation analysis

## Insights
- High magnitude earthquakes have higher chances of tsunami events.
- Most earthquakes occur in specific geographic regions.
- Magnitude and depth show some correlation.

## Conclusion
Exploratory Data Analysis helps identify patterns in earthquake data and understand factors related to tsunami occurrences.
